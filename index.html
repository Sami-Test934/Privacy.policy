Datenschutzerklärung für den Discord-Bot "BadgeBot"

1. Welche Daten werden gespeichert?

  • Discord-Benutzer-ID
  • Zugewiesene Badges
  • Datum der Badge-Vergabe

2. Wofür werden diese Daten genutzt?

Die Daten werden ausschließlich verwendet, um Badges auf dem Discord-Server anzuzeigen, zu verwalten und zu vergeben.

3. Werden Daten weitergegeben?

Nein. Alle Daten bleiben lokal in der Bot-Datenbank gespeichert und werden nicht an Dritte weitergegeben.

4. Wie lange werden die Daten gespeichert?

Die Daten werden so lange gespeichert, bis der Server oder die Badges gelöscht werden. Auf Anfrage kann ein Server-Administrator alle gespeicherten Daten löschen.

5. Ansprechpartner

Bei Fragen oder Löschanfragen bitte an deine E-Mail-Adresse oder Discord-Tag wenden.
